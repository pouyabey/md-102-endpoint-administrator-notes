# MD-102 Endpoint Administrator Study Notes

This repository contains my personal study notes and hands-on practice for the MD-102: Endpoint Administrator certification exam.

I started studying for the MD-102 exam recently, and I will keep updating this repository as I learn more topics related to Microsoft Intune, Microsoft Entra ID, Windows device management, endpoint security, and automation.

The goal of this repository is to document my learning process in a clear and practical way. I am using this repo as both a study reference and a portfolio to show my progress in endpoint administration.

---

## Course Topics I Am Studying

The topics I am studying for the MD-102 exam include:

1. Setting up the lab environment for hands-on practice
2. Adding devices to Microsoft Entra ID
3. Enrolling devices to Microsoft Intune
4. Implementing identity and compliance
5. Deploying and upgrading Windows clients using cloud-based tools
6. Planning and implementing device configuration profiles
7. Implementing Intune Suite add-on capabilities
8. Performing remote actions on devices
9. Configuring endpoint security
10. Managing device updates
11. Deploying and updating apps
12. Planning and implementing app protection and app configuration policies
13. Automating management tasks
14. Practicing real-world role-play scenarios
15. Reviewing final study steps and next steps after the course
